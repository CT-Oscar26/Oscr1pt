# Sistema de Catálogo y Carrito para Tienda de Calzado

Desarrollé una plataforma web para la gestión y exhibición de productos de una tienda de calzado, integrando PHP, MySQL y una interfaz construida con Skeleton CSS. El sistema permite mostrar productos paginados, realizar búsquedas dinámicas y visualizar información detallada de cada artículo, incluyendo imágenes, precios y marcas.

Implementé un carrito interactivo que permite agregar, modificar y vaciar productos en tiempo real mediante JavaScript. También estructuré un módulo de administración para gestionar el inventario, acompañado de un diseño limpio, funcional y adaptable a distintos dispositivos.

Este proyecto me permitió fortalecer habilidades en arquitectura web, consultas optimizadas, diseño responsivo y la integración entre frontend y backend para crear experiencias completas y fluidas.
